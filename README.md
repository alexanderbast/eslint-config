# @alexanderbast/eslint-config

> Personal eslint config. Very strict

## Install

Run `npm install --save-dev @alexanderbast/eslint-config eslint eslint-config-airbnb eslint-config-standard eslint-config-standard-jsx eslint-plugin-flowtype eslint-plugin-fp eslint-plugin-import eslint-plugin-jsx-a11y@^5.1.1 eslint-plugin-node eslint-plugin-promise eslint-plugin-react eslint-plugin-react-native eslint-plugin-standard`

## Use

Put

    "extends": [
      "@alexanderbast/eslint-config"
    ]

in your .eslintrc.json.
